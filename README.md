## 👨‍💻 «Union Smart Technology»
Multi-page website of a Slovak IT company

## 🖥️ Deployment
To deploy this project run

```
$ npm install
$ npm run dev
```

## 🛠 Skills
- Pug (Jade).
- Sass (SCSS syntax).
- JavaScript.
- Gulp.
- Swiper.
- Custom fonts.
